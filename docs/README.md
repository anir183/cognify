# Cognify 🎓🧠  
**AI-Powered Personalized & Gamified Learning Platform**

> [!IMPORTANT]  
> This is the "original" in the sense that, this was the state of the application when we submitted this to the Google Developer Group Kolkata's student hackathon "TechSprint" and bagged the 2nd prize.
> We may further develop on this idea and fix, improve or change some functionality. But this branch also keeps the first real MVP of this application.

Cognify is an AI-driven EdTech platform that delivers **truly personalized learning** through **real-time adaptation**, **performance analysis**, and **gamified experiences**.  
It moves beyond static course structures to dynamically adjust **difficulty, pace, and reinforcement** based on each learner’s performance.

---

## 🚩 Problem Statement

Most EdTech platforms follow a **one-size-fits-all** approach.  
Students with different learning styles, speeds, and strengths are forced into the same rigid structure, resulting in:

- Low engagement  
- Poor knowledge retention  
- Ineffective learning outcomes  

---

## 💡 Our Solution

**Cognify** solves this by combining:

- **AI-based performance analysis**
- **Dynamic difficulty & pacing**
- **Gamification inspired by competitive mechanics**
- **Actionable insights for educators**

The platform adapts continuously to how a student learns — not the other way around.

---

## ✨ Key Differentiators

### 🔹 AI Personalization + Gamification
- Adaptive AI meets game mechanics
- Pokémon-style **MCQ battle system** for engagement

### 🔹 Dynamic Adaptation
- Real-time difficulty adjustments
- Ensures optimal challenge (not too easy, not overwhelming)

### 🔹 Weak-Point Detection
- AI-generated summaries
- Intelligent identification of learning gaps

### 🔹 Educator-Focused Analytics
- Data-driven insights
- Visibility into common student roadblocks

---

## 🎮 Core Features

### 👨‍🎓 For Students (MVP)
- Gamified learning environment  
- AI-based learning personalization  
- Content viewing optimization  
- Difficulty modifier  
- Personalized reinforcement timelines  
- AI summarization & weak-point identification  
- Built-in AI Assistant  

### 👩‍🏫 For Educators
- **Custom Game Editor** (auto-generates gamification)
- Content editor (text & video)
- Student tracking dashboard
- AI-generated summaries & performance stats
- Identification of common learning bottlenecks
- AI-based custom certification generation
